# csv-llm-ollama

Streamlit app that analyzes a credit-card CSV and lets you ask questions in
plain English, **fully local** via Ollama.

- Port: **8504**
- LLM: `llama3.1:8b` (default, editable in the sidebar)
- DB: local SQLite (`.cache/csv-llm-ollama.sqlite`)
- Privacy: your data never leaves your machine.

## Prerequisites

```powershell
# 1. Ollama
winget install Ollama.Ollama       # or https://ollama.com/download
ollama serve                        # leave running in a window

# 2. Model (tool-calling capable)
ollama pull llama3.1:8b

# 3. Python deps (from the repo root)
pip install -r csv-llm-ollama/requirements.txt
```

## Run

```powershell
# Short way
.\csv-llm-ollama\start.ps1

# Manual
streamlit run csv-llm-ollama/app.py --server.port 8504
```

Open http://127.0.0.1:8504.

## First import

In the sidebar the default path is
`site/data/data1-anonymized.csv` (734 anonymized lines). Click
**(Re)charger ce CSV**: 733 transactions are loaded into the `data1`
profile.

## Sample questions

- *How much did I spend on groceries last month?*
- *What is my biggest spending category?*
- *How many times did I visit Coffee Gossip this year and how much did I spend?*
- *List my top 5 merchants.*

Each answer shows:
- the SQL query generated by the model,
- the rows returned by SQLite,
- the natural-language answer.
